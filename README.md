# Dmitry Glukhov

I build practical AI-agent workflows and local-first tools: macOS utilities, Telegram/LLM systems, translation pipelines, and developer automation around Codex, Gemini, and OpenAI-compatible providers.

## Current Focus

- AI-assisted software delivery: specs, architecture, agent task decomposition, code review, tests, and release-ready handoff.
- Local-first LLM systems: SQL-backed knowledge pipelines, Telegram data processing, provider routing, and runtime control centers.
- macOS AI utilities: native Swift apps for live audio translation, OCR subtitles, menu bar workflows, and local automation.
- Translation and transcription tooling: OCR, localization QA, back-translation checks, audio/video preparation, and Markdown export pipelines.

## How I Work With Agents

- Decompose real product work into scoped implementation, review, test, and release steps.
- Delegate bounded coding or verification tasks to faster/weaker agents when the contract is clear.
- Keep stronger models on architecture, integration review, failure analysis, and final judgment.
- Use guardrails: git/worktree checks, pipeline status checks, lint/test hooks, and small reviewable commits.
